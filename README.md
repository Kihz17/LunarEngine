INSTALLING:
- Run Installer.exe
- A folder on your desktop should be created called "Lunar"
- Open the folder and run "Project1.exe"

CONTROLS:
WASD to move camera
Mouse to Look
ESC to toggle mouse cursor
LMB to shoot

COLORS:
Evade - Blue
Flee - Yellow
Idle - White
Pursue - Red
Seek - Purple
Wander - Green
